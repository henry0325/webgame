# 諧音地下城：節奏遠征

可直接在瀏覽器執行的惡搞節奏戰鬥網頁遊戲。

## 特色

- 諧音技能 + TRPG 回合戰鬥
- 可調 BPM（70~160）與節拍音，BPM 越高風險越高、傷害倍率越高
- 多波怪物連戰（共 5 隻）+ 挑戰係數隨關卡成長
- 背包/裝備/戰鬥融合：怪物掉落符文，裝備後直接加成技能與舒適度
- 舒適度系統：影響傷害、防禦與畫面穩定度
- 持續成長：等級、經驗、總擊敗數、永久加成 + 天賦點（localStorage）
- 模式化設計：標準遠征 + 每日挑戰（固定種子）
- 任務與分數系統：連擊、Perfect、整理、舒適度、擊殺進度都會累積分數
- 先行美術：內建 player / enemy SVG 角色圖 + 打擊動畫 pulse，可直接替換擴充

## 快速開始

```bash
npm test
npm start
```

啟動後開啟：`http://localhost:8080`

## 操作

- `1 / 2 / 3`：選技能
- `空白鍵` 或 `Enter`：在拍點出招
- `BPM 滑桿`：調整節奏速度（影響速度倍率）
- `開啟節拍音`：播放對應 BPM 的節拍聲
- 點擊背包物品：裝備符文加成
- `一鍵整齊收納`：排序並提升舒適度
- `標準遠征 / 每日挑戰`：切換遊玩模式


## 新增機制

- Boss 蓄力攻擊：每 3 回合觸發一次重擊，需靠節奏與防禦天賦應對。
- 天賦樹：
  - 節拍專注：放寬判定窗。
  - 抗壓心流：降低敵方傷害。
  - 掉寶手感：提升掉落裝備攻擊值。
