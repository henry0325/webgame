# 諧音地下城：節奏遠征

可直接在瀏覽器執行的惡搞節奏戰鬥網頁遊戲。

## 特色

- 諧音技能 + TRPG 回合戰鬥
- 可調 BPM（預設放慢到 90）與可切換節拍音
- 多波怪物連戰（共 5 隻）
- 背包/裝備/戰鬥融合：怪物掉落符文，裝備後直接加成技能
- 舒適度系統：影響傷害、防禦與畫面穩定度
- 模式化設計：標準遠征 + 每日挑戰（固定種子）
- 任務與分數系統：連擊、Perfect、整理、舒適度、擊殺進度都會累積分數

## 快速開始

```bash
npm test
npm start
```

啟動後開啟：`http://localhost:8080`

## 操作

- `1 / 2 / 3`：選技能
- `空白鍵` 或 `Enter`：在拍點出招
- `BPM 滑桿`：調整節奏速度
- `開啟節拍音`：播放對應 BPM 的節拍聲
- 點擊背包物品：裝備符文加成
- `一鍵整齊收納`：排序並提升舒適度
- `標準遠征 / 每日挑戰`：切換遊玩模式
