# Sora 美術素材提示詞（山海經 × 詼諧）

將輸出檔案命名為：
- `assets/player_sora.png`
- `assets/enemy_sora.png`

遊戲會自動優先讀取這兩張 PNG；若不存在則回退到 SVG。

---

## 1) 玩家立繪（player_sora.png）

**Prompt (zh-TW):**

山海經風格角色立繪，古卷紙質感與墨線，角色是「鹿角魚尾、戴墨鏡、嘴角微笑」的奇獸英雄，帶有幽默感但不低幼。配色以青藍、赭石、象牙白為主，局部紅色符咒點綴。構圖為半身正面，留白乾淨，2D 插畫風，線條清晰、可做遊戲 UI 頭像。避免寫實恐怖、避免過度擬真。高質感、可商用風格。

**Negative prompt:**
低解析、模糊、比例崩壞、過度血腥、恐怖獵奇、文字浮水印、LOGO。

---

## 2) 敵人立繪（enemy_sora.png）

**Prompt (zh-TW):**

山海經怪物立繪，虎身鳥翼的異獸，手拿珍奶作為詼諧元素，整體氣勢強但帶喜感。古卷紙張底紋與墨線描邊，主色紫褐、暗紅、金黃，眼神銳利，線條俐落，2D 遊戲角色插畫。構圖半身正面，背景簡化，適合 UI 頭像裁切。避免可愛卡通化過度。

**Negative prompt:**
Q版幼兒風、低品質、過多背景雜訊、文字浮水印、血腥內臟。

---

## 建議輸出規格
- 尺寸：`1024x1024`
- 格式：`PNG`
- 背景：建議有淡底紋，不要透明（避免 UI 合成太單薄）
